# M1: Warp i18n 基础设施 + 语言开关 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 搭建 Warp 的 i18n 基础设施（rust-i18n + sys-locale），在 Appearance 设置页加 Language 下拉开关（System / English / 简体中文），启动时根据系统语言自动选择，切换无需重启。配套 10 个示例 zh-CN 翻译验证整条链路。

**Architecture:** 新建 `crates/i18n` 作为单一入口（暴露 `t!` 宏、`AppLanguage` 枚举、`detect_system_language`、`apply` 函数），rust-i18n 编译期嵌入 `_locales/zh-CN/*.yml`。`app/src/settings/language.rs` 仿照 `font.rs` 用 `define_settings_group!` 定义 `LanguageSettings`。`appearance_page.rs` 1:1 复刻 `font_weight_dropdown` 完整链路（字段→构造→Action→dispatch→写回→渲染）。启动序列在 `register_all_settings` 后调用 `i18n::init::apply()`。

**Tech Stack:** Rust 2024 edition, rust-i18n 3.x, sys-locale 0.3.x, Warp 自研 settings 宏（`define_settings_group!` / `SettingsValue` derive）

**Reference Spec:** `docs/superpowers/specs/2026-06-19-warp-i18n-zh-cn-design.md`

**Environment Note:** 当前开发机无 cargo/rustc（见 `which cargo` 无输出）。本计划写好后需在装有 Rust 工具链的环境执行。所有命令都假设工作目录为 `/root/DEV/warp`。

```rust
//! 系统语言检测。
//!
//! 把跨平台 OS API（`sys_locale::get_locale()`）返回的多种格式统一归一化到
//! Warp 内部使用的 BCP-47 代码（目前仅 `"zh-CN"` 与 `"en"`）。
//!
//! 归一化结果在进程生命周期内通过 [`DETECTED`] 缓存一次，
//! 仅当用户设置选择 `System` 时才会触发首次检测。

use std::sync::OnceLock;

static DETECTED: OnceLock<&'static str> = OnceLock::new();

/// 读取系统语言并归一化。结果缓存。
///
/// 返回值是 rust-i18n 能识别的 locale 字符串：
/// - 中文系统（含简/繁）→ `"zh-CN"`
/// - 其他系统 → `"en"`
pub fn detect_system_language() -> &'static str {
    *DETECTED.get_or_init(|| {
        let raw = sys_locale::get_locale().unwrap_or_else(|| "en".to_string());
        normalize(&raw).unwrap_or("en")
    })
}

/// 把各种 locale 字符串归一化为 Warp 内部使用的 BCP-47 代码。
///
/// 支持的输入格式：
/// - POSIX: `zh_CN.UTF-8`, `zh_CN`, `en_US`
/// - macOS NSLocale: `zh-Hans-CN`, `zh-Hant-TW`
/// - Windows / 通用 BCP-47: `zh-CN`, `en-US`
///
/// 返回 `None` 表示输入为空字符串，调用方应 fallback 到 `"en"`。
pub fn normalize(raw: &str) -> Option<&'static str> {
    let trimmed = raw.trim();
    if trimmed.is_empty() {
        return None;
    }
    let lower = trimmed.to_lowercase();
    if lower.starts_with("zh") {
        // MVP：所有中文变体（简/繁/区域）都归一到 zh-CN
        Some("zh-CN")
    } else {
        // 其他语言一律 fallback 到英文（MVP 范围只覆盖中英）
        Some("en")
    }
}
```

> 注意：spec 中提到的"繁中检测特殊路径"在 MVP 简化为"所有 zh* → zh-CN"。归一化函数签名保持 `Option`，方便后续扩展精确区分。

- [ ] **Step 4.4: 实现 `language.rs`**

替换 `/root/DEV/warp/crates/i18n/src/language.rs` 全部内容：

```rust
//! 用户可选的应用语言枚举。
//!
//! 作为 `LanguageSettings`（在 app crate 中）的值类型，
//! 通过 [`settings_value::SettingsValue`] 派生宏接入 Warp 的 settings 系统，
//! 序列化为 snake_case（参考 `app/src/settings/mod.rs:460-484` 的 `EnforceMinimumContrast`）。

use serde::{Deserialize, Serialize};

/// 用户可选的应用界面语言。
///
/// 顺序决定下拉项的展示顺序（参考 [`AppLanguage::all`]）。
#[derive(
    Clone,
    Copy,
    Debug,
    Default,
    PartialEq,
    Eq,
    Serialize,
    Deserialize,
    schemars::JsonSchema,
    settings_value::SettingsValue,
)]
#[schemars(
    description = "The display language of the Warp UI.",
    rename_all = "snake_case"
)]
pub enum AppLanguage {
    /// 跟随系统语言（启动时检测一次）。
    #[default]
    System,
    /// 强制英文。
    English,
    /// 强制简体中文。
    SimplifiedChinese,
}

impl AppLanguage {
    /// 下拉项的可选值列表（顺序即 UI 展示顺序）。
    pub fn all() -> &'static [AppLanguage] {
        &[Self::System, Self::English, Self::SimplifiedChinese]
    }

    /// 下拉项的显示文案。
    ///
    /// 约定：非英文选项用目标语言的母语写法显示，
    /// 方便用户在英文 UI 下也能识别自己的语言。
    pub fn dropdown_label(&self) -> &'static str {
        match self {
            Self::System => "System",
            Self::English => "English",
            Self::SimplifiedChinese => "简体中文",
        }
    }

    /// 解析为 rust-i18n 能识别的 BCP-47 语言代码。
    ///
    /// - [`AppLanguage::System`] 会触发 [`crate::detection::detect_system_language`]，
    ///   返回值在 MVP 中只可能是 `"zh-CN"` 或 `"en"`。
    /// - [`AppLanguage::English`] / [`AppLanguage::SimplifiedChinese`] 返回固定值。
    pub fn resolve(self) -> &'static str {
        match self {
            Self::System => crate::detection::detect_system_language(),
            Self::English => "en",
            Self::SimplifiedChinese => "zh-CN",
        }
    }
```
